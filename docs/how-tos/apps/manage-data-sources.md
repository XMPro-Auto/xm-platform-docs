# Manage Data Sources

Data Sources can be created for a Page in the Application. They are a way to link to a specific Entity in a Connection's Entities, for example, a table in a SQL database. Data Sources are managed through the Page Data tab of a Page. Data Sources allow you to display, use, manipulate or store data from a connected source of data such as an SQL Server Database.

> [!NOTE]
> It is recommended that you read the article listed below to improve your understanding of Data Integration.
>
> * [Data Integration](../../concepts/application/data-integration.md)
> * [How to Manage Connections](manage-connections.md)

## Creating a Data Source

To create a Data Source on the Page of an Application, follow the steps below:

1. Open the Editor for the Application.

   ![Screenshot of the application dashboard showing how to open the App Editor with the edit button highlighted](../../images/apps/datasources/edit-app-editor.png)

2. Open the Page that will have the Data Source.

   ![Screenshot showing how to open a page from the pages panel in the App Editor](../../images/apps/datasources/open-page.png)

3. Click on _Page Data_.
4. Click on the _plus_ symbol to add a new Data Source.
5. Enter a name.
6. Add the Connection and Entity where data will be retrieved.
7. Specify the [Primary Key](../../concepts/application/data-integration.md#primary-key).
8. Click on _Save_.

   ![Screenshot showing the primary key field in the Data Source configuration panel](../../images/apps/datasources/primary-key.png)

   ![Screenshot of the Data Source creation interface with connection, entity, and primary key fields](../../images/apps/datasources/create-datasource.png)

## Edit a Data Source

To edit a Data Source, follow the steps below:

1. Open the Editor for the Application.

   ![Screenshot of the application dashboard showing how to open the App Editor for editing a Data Source](../../images/apps/datasources/edit-app-editor.png)

2. Open the Page that has the Data Source.

   ![Screenshot showing how to open a page containing the Data Source you want to edit](../../images/apps/datasources/edit-open-page.png)

3. Click on _Page Data_.
4. Click on the _edit_ button.
5. Edit the details of the Data Source.
6. Click on _Save_.

   ![Screenshot showing the Data Source edit interface with editable fields](../../images/apps/datasources/edit-datasource.png)

## Delete a Data Source

To delete a Data Source, follow the steps below:

1. Open the Editor for the Application.

   ![Screenshot of the application dashboard showing how to open the App Editor for deleting a Data Source](../../images/apps/datasources/delete-app-editor.png)

2. Open the Page that has the Data Source.

   ![Screenshot showing how to open a page containing the Data Source you want to delete](../../images/apps/datasources/delete-open-page.png)

3. Click on _Page Data_.
4. Click on the _edit_ button.
5. Click on _Delete_.
6. Confirm that you want to delete the Data Source.

   ![Screenshot showing the delete button and confirmation dialog for removing a Data Source](../../images/apps/datasources/delete-datasource.png)
